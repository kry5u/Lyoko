# Projekt na przedmiot programowanie zaawansowane
## Instrukcje pierwszego uruchomienia

Uruchomić Visual Studio 2022.  

Wybrać opcję „Klonuj repozytorium”  

![alt text](https://github.com/kry5u/Lyoko/blob/main/Lyoko/wwwroot/images/1.PNG "1")  

Wprowadzić ścieżkę do repozytorium na GitHubie.  

> [https://github.com/kry5u  ](https://github.com/kry5u/Lyoko)

Program automatycznie utworzy folder zgodny z nazwą projektu.  

![alt text](https://github.com/kry5u/Lyoko/blob/main/Lyoko/wwwroot/images/2.jpg "2")  

> Otworzenie spowoduje pobranie wszystkich niezbędnych pakietów NuGet w lewym dolnym oknie programu.

![alt text](https://github.com/kry5u/Lyoko/blob/main/Lyoko/wwwroot/images/3.jpg "3")  

Uruchomić projekt za pomocą F5 lub klikając przycisk https.  

![alt text](https://github.com/kry5u/Lyoko/blob/main/Lyoko/wwwroot/images/4.PNG "4")  

> Podczas uruchamiania programu, może zostać wyświetlona informacja o certyfikatach SSL. Można ją zaakceptować, aby uniknąć ostrzeżeń SSL w przeglądarce.  

![alt text](https://github.com/kry5u/Lyoko/blob/main/Lyoko/wwwroot/images/5.PNG "5")
